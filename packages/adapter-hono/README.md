# @fonderie-js/adapter-hono

Hono adapter for fonderie-js — bridge(), adapt(), mount() to use fonderie middleware in native Hono routes.

## Install

```sh
npm install @fonderie-js/adapter-hono
```

## Use

```ts
import { OPERATIONS, adapt, bridge } from '@fonderie-js/adapter-hono';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
