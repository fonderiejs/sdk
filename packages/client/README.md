# @fonderie-js/client

Isomorphic TypeScript client for Fonderie-powered APIs — fully typed request and response shapes, zero runtime dependencies.

## Install

```sh
npm install @fonderie-js/client
```

## Use

```ts
import { AuthClient, FonderieApiError, FonderieClient } from '@fonderie-js/client';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
