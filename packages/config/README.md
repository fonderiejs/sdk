# @fonderie-js/config

DB-backed feature flags and remote config — per-environment overrides, TTL-based hot reload, and a typed getConfig helper usable inside any middleware.

## Install

```sh
npm install @fonderie-js/config
```

## Use

```ts
import { CONFIG_MANAGER_KEY, configContextMiddleware, getConfig } from '@fonderie-js/config';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
