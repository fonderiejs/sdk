# @fonderie-js/core

Framework core — request router, middleware pipeline, module system, and shared context types. Every other @fonderie-js package depends on this.

## Install

```sh
npm install @fonderie-js/core
```

## Use

```ts
import { defineConfig, arrayOrEmpty, booleanOrFalse } from '@fonderie-js/core';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
