# @fonderie-js/audit

Workspace-scoped audit log — query fonderie_events for a human-readable activity trail.

## Install

```sh
npm install @fonderie-js/audit
```

## Use

```ts
import { AuditModule } from '@fonderie-js/audit';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
