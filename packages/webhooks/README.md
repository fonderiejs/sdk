# @fonderie-js/webhooks

Outgoing webhook engine — register endpoints, fan out workspace-scoped events, track delivery, and retry on failure.

## Install

```sh
npm install @fonderie-js/webhooks
```

## Use

```ts
import { WebhooksModule } from '@fonderie-js/webhooks';
```

Part of [Fonderie](https://fonderie.ai) — the software foundry. Monorepo, docs, and issues live at [fonderie-js/sdk](https://github.com/fonderie-js/sdk). Follow [@fonderiejs](https://x.com/fonderiejs).
